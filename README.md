# pcb_ruler_light
pcb ruler with mcu and lights on it

pcb的尺子

长30cm，宽4cm

有51的最小系统和32个LED，可以做手摇棒，利用视觉暂留效果，显示文字

目前只开放Gerber文件
因为＞﹏＜pcb文件在旧电脑上
